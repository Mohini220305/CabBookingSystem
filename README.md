# Cab Booking System

A C-based Cab Booking System with a web frontend, developed using modular programming and file handling, featuring Dijkstra’s algorithm for route management.

## Features

- Multi-role login: Admin, Driver, Customer
- Ride booking and assignment
- Ride reassignment
- Stack-based billing
- Ride history tracking
- Web-based frontend for interactive use

## Technologies

- C (Backend)
- HTML/CSS, JavaScript (Frontend)
- File handling for persistent storage
- Dijkstra’s algorithm for shortest route calculation
